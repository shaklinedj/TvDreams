│  Gestión básica:                                                         │
│  ─────────────                                                           │
│  docker-compose up -d              # Iniciar                             │
│  docker-compose down               # Detener                             │

│  Monitoreo:                                                              │
│  ──────────                                                              │
│  docker-compose ps                 # Ver estado                          │
│  docker-compose logs -f app        # Logs en vivo (Ctrl+C)              │
│  docker-compose logs --tail 50     # Últimas 50 líneas                  │
│  docker stats                      # Uso recursos real-time             │
│                                                                            │
│  Base de datos:                                                          │
│  ──────────────                                                          │
│  docker-compose exec db mysql -u cms_user -p  # CLI MySQL              │
│  docker-compose exec app npm run build         # Rebuild frontend       │

│     ──────────────────────────────────                                   │

│       up -d                                                              │
│                                                                            │
│  3. SSL con Let's Encrypt:                                               │
│     ──────────────────────                                               │
│     sudo certbot certonly --nginx -d tvdreams.tudominio.com             │
│     (Copiar config desde nginx-tvdreams.conf)                           │
│                                                                            │
│  4. Monitoring:                                                           │
│     ──────────                                                           │
