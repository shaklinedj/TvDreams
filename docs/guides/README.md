# 📚 Documentación y Guías - CMS HLAURE

Esta carpeta contiene todas las guías y documentación detallada del sistema CMS HLAURE.

## 📋 Guías Disponibles

### 🚀 **[DEPLOY.md](./DEPLOY.md)**
- Guía completa de despliegue en producción
- Configuración con Laragon, XAMPP y PHP/MySQL nativo

- WebSocket support

### ⚡ **[GUIA-RAPIDA-CONFIGURACION.md](./GUIA-RAPIDA-CONFIGURACION.md)**
- Guía express para cambios de host/IP
- Configuración rápida de red
- Variables de entorno esenciales

### 📁 **[TIPOS-ARCHIVOS-SOPORTADOS.md](./TIPOS-ARCHIVOS-SOPORTADOS.md)**

- Límites de tamaño y especificaciones
- Optimización de archivos

### 📖 **[RESUMEN-APLICACION.md](./RESUMEN-APLICACION.md)**
- Resumen técnico completo de la aplicación
- Stack tecnológico y arquitectura
- Base de datos y relaciones
- APIs y endpoints

## 🎯 Uso Recomendado

1. **Para desarrolladores nuevos**: Comenzar con `RESUMEN-APLICACION.md`
2. **Para despliegue rápido**: Seguir `DEPLOY.md`
3. **Para configuración de red**: Usar `GUIA-RAPIDA-CONFIGURACION.md`
4. **Para producción con dominio**: Implementar `REVERSE-PROXY.md`
